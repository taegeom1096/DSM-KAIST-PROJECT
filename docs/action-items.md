## 액션 아이템 1
- 사용자 액션: 새로운 추천 메뉴를 등록한다
- 서버가 해야 할 일: 새 메뉴 정보를 저장한다
- 저장할 데이터: 메뉴이름, 카테고리(한식/중식 등), 맵기단계, 등록자명
- 후보 테이블: recommended_menus

## 액션 아이템 2
- 사용자 액션: 전체 메뉴 목록을 조회한다
- 서버가 해야 할 일: 저장된 메뉴 목록을 가져온다
- 저장할 데이터: 저장된 메뉴 리스트
- 후보 테이블: recommended_menus

## 액션 아이템 3
- 사용자 액션: 오늘의 메뉴를 랜덤 추첨한다
- 서버가 해야 할 일: 저장된 메뉴 중 무작위로 하나를 선택하여 반환한다
- 저장할 데이터: 추첨된 메뉴 정보
- 후보 테이블: recommended_menus

## 액션 아이템 4
- 사용자 액션: 추천된 메뉴에 한 줄 평(댓글)을 작성한다
- 서버가 해야 할 일: 특정 메뉴에 유저의 평가를 저장한다
- 저장할 데이터: 메뉴 id, 작성자명, 한 줄 평 내용, 별점
- 후보 테이블: menu_reviews

## 액션 아이템 5
- 사용자 액션: 작성한 한 줄 평을 삭제한다
- 서버가 해야 할 일: 특정 한 줄 평을 삭제한다
- 저장할 데이터: 리뷰 id
- 후보 테이블: menu_reviews
